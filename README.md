# vue3.2+ts+elementPlus+vite4
# 纯前端管理系统模板
# 可以直接拿来做启动项目

在线预览后台管理 [http://vue3.88an.top](http://vue3.88an.top)

# 内置如下功能：
## 暗黑模式切换
## 主题切换
## 多语言切换
## 路由切换动画
## 拖拽排序
## 图片管理组件
## 百度富文本编辑器
## 富文本编辑器模板功能
## 路由tag
## 富文本编辑器图片使用图片管理组件

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
